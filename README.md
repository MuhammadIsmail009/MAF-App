AI-Based Memory Forensics Analyzer - Frontend
============================================

Next.js 14 (App Router) dashboard for the AI-Based Memory Forensics Analyzer.

Key tech:
- Next.js 14
- TypeScript + React
- TailwindCSS
- shadcn-style UI components
- React Query (TanStack Query)
- Axios
- React Flow / D3-based visualizations

Run locally (after installing dependencies):

```bash
cd frontend
npm install
npm run dev
```
